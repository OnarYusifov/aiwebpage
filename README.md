# Boldo Website - Customized Business Template

A modern, responsive website built from the Boldo template with extensive customizations.

## Features

- **Modern Design**: Clean, professional layout with purple-to-teal gradient theme
- **Responsive**: Fully responsive design that works on all devices
- **Multi-page**: Home, About, Blogs, and Contact pages
- **Social Media Integration**: LinkedIn, Twitter, Facebook, and Instagram links
- **Contact Form**: Functional contact form on the Contact page
- **Custom Styling**: Extensive CSS customizations for unique look

## Pages

- `index.html` - Home page with hero section and services
- `about.html` - About page with company information
- `blogs.html` - Blog listing page
- `contact.html` - Contact page with form and information

## Customizations Made

1. **Color Scheme**: Changed from green to purple/teal gradient theme
2. **Navigation**: Added Contact page to navigation menu
3. **Footer**: Updated with social media links and proper navigation
4. **Content**: Customized text and descriptions throughout
5. **Buttons**: Updated to use gradient backgrounds
6. **Styling**: Enhanced with custom CSS animations and hover effects

## Technologies Used

- HTML5
- CSS3 (Custom styles in `assets/css/user.css`)
- Bootstrap 5
- Font Awesome Icons
- JavaScript

## Hosting

This website is hosted on GitHub Pages. To deploy:

1. Push code to GitHub repository
2. Go to repository Settings > Pages
3. Select main branch and / (root) folder
4. Your site will be live at: `https://[username].github.io/[repository-name]`

## Local Development

1. Clone the repository
2. Open `index.html` in a web browser
3. All files are static HTML/CSS/JS - no build process required
